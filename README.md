# MediQueue – Tutor Booking System

A modern full-stack tutor booking platform connecting students with expert educators.

🌐 **Live Site:** [https://mediqueue-client-eight.vercel.app](https://mediqueue-client-eight.vercel.app)

## ✨ Features

- 🔐 **Firebase Authentication** – Email/password and Google login with JWT-secured private routes
- 🎓 **Browse & Book Tutors** – Explore tutors by subject, filter by date, search by name
- 📅 **Smart Booking System** – Slot-limited sessions with date restrictions and auto-decrement
- 🧑‍🏫 **Tutor Management** – Add, edit, and delete your own tutor profiles
- 🌙 **Dark / Light Mode** – System-aware theme toggle across the entire app
- 📱 **Fully Responsive** – Optimized for mobile, tablet, and desktop screens

## 🛠 Tech Stack

- **Frontend:** React 18, Vite, Tailwind CSS, DaisyUI
- **Backend:** Node.js, Express
- **Database:** MongoDB Atlas
- **Auth:** Firebase + JWT
- **Hosting:** Vercel

## 🚀 Getting Started

```bash
npm install
cp .env.example .env  # fill in your Firebase & API keys
npm run dev
```
